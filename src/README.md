# 🤖 Telegram Automation Bot

A Python-based Telegram bot that sends automated messages using the Telegram Bot API.

---

## 🚀 Features
- Sends automated messages to Telegram chat
- Uses Telegram Bot API
- Simple HTTP request-based automation
- Beginner-friendly bot project

---

## 🛠️ Tech Stack
- Python 3
- Requests library
- Telegram Bot API

---

## ⚙️ How It Works
1. Connects to Telegram Bot API  
2. Sends POST request with message data  
3. Telegram delivers message to chat  

---

## ▶️ How to Run
```bash
pip install -r requirements.txt
python src/main.py
```
